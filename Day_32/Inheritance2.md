# Inheritance Day 2
1. Hospital Management System
Description

Create a hospital management system using inheritance.

Create a parent class:

Person

Child classes:

Doctor
Nurse
Patient

The parent class should contain common properties:

name
age
gender

Each child class should have its own additional features.

Examples:

Doctor → specialization
Nurse → wardAssigned
Patient → disease

Add methods:

displayInfo()
performDuty()

This question helps you learn:

Inheritance hierarchy
Code reusability
Method overriding
Real-world OOP design
Example 1
Input:
Doctor Name = Rahul
Specialization = Cardiologist

Output:
Doctor Rahul is treating patients
Example 2
Input:
Nurse Name = Priya
Ward = ICU

Output:
Nurse Priya is taking care of ICU patients
Example 3
Input:
Patient Name = Aman
Disease = Fever

Output:
Patient Aman is suffering from Fever
2. Online Shopping System
Description

Create a parent class:

Product

Child classes:

Electronics
Clothing
Food

The parent class should contain:

productName
price
quantity

Each child class should contain its own features.

Examples:

Electronics → warranty
Clothing → size

Hierarchical inheritance
Method overriding
Real-world business logic

3. University Course Management System
Description

Design a university course management system using inheritance.

Create a parent class:

Course

Child classes:

ProgrammingCourse
MathCourse
ScienceCourse

Common properties:

courseName
duration
instructor

Each course should have specific functionality.

Examples:

Programming → coding language
Math → formula count
Science → laboratory required

Add methods:

showCourseDetails()
startCourse()

This question helps you understand:

Inheritance structure
Specialized behavior
Real-world educational systems
Example 1
Input:
Programming Course = Java

Output:
Java Programming Course Started
Example 2
Input:
Math Course = Algebra

Output:
Algebra Math Course Started
Example 3
Input:
Science Course = Physics

Physics Science Lab Session Started
