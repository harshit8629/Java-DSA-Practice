1. Divide Two Numbers (Handle ArithmeticException)

You are given two integers a and b.

Return the result of division. If b = 0, handle the exception and print "Cannot divide by zero".

Example:

Input:
a = 10, b = 0

Output:
Cannot divide by zero

2. Array Access (Handle ArrayIndexOutOfBoundsException)

You are given an array of integers and an index.

Return the element at that index. If the index is invalid, handle the exception.

Example:

Input:
arr = [1, 2, 3], index = 5

Output:
Index out of bounds

3. String to Integer Conversion (Handle NumberFormatException)

You are given a string s.

Convert it into an integer. If the string is not a valid number, handle the exception.

Example:

Input:
s = "123a"

Output:
Invalid number format

4. Null String Length (Handle NullPointerException)

You are given a string s.

Print its length. If the string is null, handle the exception.

Example:

Input:
s = null

Output:
String is null
