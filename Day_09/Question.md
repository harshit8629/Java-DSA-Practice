1. Factorial of a Number

You are given an integer n.

Return the factorial of the number using recursion.

Example:

Input: n = 5
Output: 120

Explanation:

Factorial of 5 = 5 × 4 × 3 × 2 × 1 = 120

2. Fibonacci Number

You are given an integer n.

Return the nth Fibonacci number using recursion.

Example:

Input: n = 6
Output: 8

Explanation:

Fibonacci sequence: 0, 1, 1, 2, 3, 5, 8

3. Sum of First N Numbers

You are given an integer n.

Return the sum of first n natural numbers using recursion.

Example:

Input: n = 5
Output: 15

Explanation:

1 + 2 + 3 + 4 + 5 = 15

4. Reverse a String

You are given a string s.

Return the reversed string using recursion.

Example:

Input: s = "hello"
Output: "olleh"

Explanation:

Reversing the string gives "olleh"
