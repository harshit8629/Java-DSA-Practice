Problem 1.1

Write a function lastThree() that takes a string as input and returns a string containing only the last three characters from the input string.
Assume input string has at least 3 characters.

Input | Return Value
lastThree("abcdef") | "def"
lastThree("morning") | "ing"

Problem 1.2

Write a function printLastThree() that takes a list of strings as input and prints the last three characters of every string.

Note: Use lastThree() function from Problem 1.1

Input | Return Value
printLastThree(["coding","python","java"]) |
  ing
  hon
  ava

Problem 2.1

Write a function countVowels() that takes a string as input and returns the number of vowels (a, e, i, o, u) in the string.

Input | Return Value
countVowels("hello") | 2
countVowels("education") | 5

Problem 2.2

Write a function printVowelCount() that takes a list of strings as input and prints the number of vowels in each string.

Note: Use countVowels() function from Problem 2.1

Input | Return Value
printVowelCount(["apple","sky","idea"]) |
  2
  0
  3

Input | Return Value

Problem 3.2
