1. Encapsulation Using Private Variable

You are given a class BankAccount with attribute balance.

Create:

a private variable balance
methods:
deposit()
withdraw()
getBalance()

Do not allow direct access to balance.

Example
Input
account.deposit(5000)
account.withdraw(2000)
Output
Balance: 3000
Explanation

The balance variable is hidden using encapsulation.

Access is controlled through methods.

2. Encapsulation for Student Marks

You are given a class Student with attribute marks.

Create:

private variable marks
method setMarks()
method getMarks()
Example
Input
student.setMarks(85)
Output
Marks: 85
Explanation

Marks cannot be changed directly outside the class.

Setter and getter methods are used.
